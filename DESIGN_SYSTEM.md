# Design System

This document outlines the design principles and standards for this application.

## Color System

### Primary Colors
- **Primary Blue**: `#1A73E8` or `#0D6EFD`
- **Secondary Dark**: `#111827`

### Neutral Colors
- **Background**: `#F9FAFB`
- **Surface**: `#FFFFFF`
- **Border**: `#E5E7EB`
- **Text Primary**: `#111827`
- **Text Secondary**: `#6B7280`
- **Text Tertiary**: `#9CA3AF`

### Accent Colors
- **Success/Green**: `#10B981`
- **Warning/Amber**: `#F59E0B`
- **Error/Red**: `#EF4444`

## Typography

- **Font Family**: Inter, Roboto, or system sans-serif
- **Headings**: Bold, clear, well-spaced
- **Body Text**: 14-16px, medium contrast
- **Line Height**: 1.5-1.6

## Spacing Scale (8px base)

- `8px` (0.5rem) - `gap-2`, `p-2`
- `16px` (1rem) - `gap-4`, `p-4`
- `24px` (1.5rem) - `gap-6`, `p-6`
- `32px` (2rem) - `gap-8`, `p-8`
- `48px` (3rem) - `gap-12`, `p-12`

## Border Radius

- **Small**: `rounded-md` (6px)
- **Medium**: `rounded-lg` (8px)
- **Large**: `rounded-xl` (12px)

## Shadows

- **Subtle**: `shadow-sm`
- **Default**: `shadow`
- **Hover**: `shadow-md`

## Transitions

- **Duration**: 150-250ms
- **Easing**: `ease-in-out`

## Components

### Button
- Primary: Blue background (#1A73E8)
- Secondary: Gray background
- Outline: Transparent with border
- Consistent padding: 12px 24px
- Smooth transitions

### Card
- White background
- Subtle border
- Rounded corners (lg)
- Padding: 24px (p-6)

### Input
- Clear labels
- Consistent sizing
- Focus states with ring
- Helper text support

### Table
- Sticky headers
- Hover states
- Sortable columns
- Pagination

## Accessibility

- ARIA labels
- Keyboard navigation
- Focus indicators
- Sufficient contrast ratios
